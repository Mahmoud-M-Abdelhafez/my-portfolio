Shoubra Store MVC

Shoubra Store is a fully functional e-commerce web application that allows users to browse products, manage their shopping experience, and interact with reviews and profiles. Built using the latest .NET Core 8 framework, this project incorporates a variety of essential features and technologies to create a seamless and secure shopping platform.

View Source Code

Project Overview

Key Features:

Login & Register Pages: Secure user authentication and authorization, ensuring that accounts are protected with advanced security mechanisms.

Product & Category Pages: Users can easily explore and filter products, with detailed views to help them make informed decisions.

Shopping Cart: A smooth experience for adding, adjusting, or removing items from the cart, making shopping more convenient.

Review Management: Users can leave reviews on products, edit them, or delete them based on their experience.

Profile Management: Personal profiles where users can update their details and profile picture.

Admin Dashboard: Admins can manage products, categories, users, reviews, and roles to keep the store running smoothly.
